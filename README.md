## Overview
This tableau dashboard analyzes personal reading habits, inlcuding genre preferences, author frequency, monthly reading trends, and average ratings across genres.

## Tools Used
- Tableau Public
- Excel

## Dashboard Features
- Most Read Genres
- Highest Rated Genres
- Most Read Authors
- Books Finished by Month

## Key Insights
- Fantasy and Fiction were the most frequently read genres.
- Fiction received the higest average ratings.
- Reading activity peaked during May.
- Sarah J. Maas appeared significantly more frequently than other authors, showing a strong preference pattern rather than an isolated outlier.
